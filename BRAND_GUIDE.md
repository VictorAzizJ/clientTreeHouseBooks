## üéØ Usage Guidelines

### When to Use Each Color

#### Primary Teal (#2B7A78)
‚úÖ **DO:**
- Main navigation bar
- Primary action buttons ("Save", "Submit", "Create")
- Page headers and titles
- Active menu items
- Important links

‚ùå **DON'T:**
- Overuse - balance with white space
- Use for warning/error states
- Use on dark backgrounds

#### Light Teal (#3AAFA9)
‚úÖ **DO:**
- Secondary buttons
- Icon colors
- Border highlights
- Hover states
- Subtle accents

‚ùå **DON'T:**
- Use as main background
- Use for text (poor contrast)

#### Orange (#FF6B35)
‚úÖ **DO:**
- "New" or "Popular" badges
- Call-to-action buttons
- Limited highlights
- Special promotions

‚ùå **DON'T:**
- Overuse - save for special emphasis
- Use as primary color
- Mix with red error states

## ‚ôø Accessibility

### Color Contrast Ratios

**Text on Backgrounds:**
- Dark Gray (#2B2B2B) on White: 14.5:1 ‚úÖ (AAA)
- Primary Teal (#2B7A78) on White: 4.7:1 ‚úÖ (AA)
- White on Primary Teal: 4.5:1 ‚úÖ (AA)
- Light Gray (#6B6B6B) on White: 5.7:1 ‚úÖ (AA)

**Do Not Use:**
- Light Teal text on white (poor contrast)
- Orange text on white (poor contrast)
- Gray text smaller than 14px

### Best Practices
- ‚úÖ Always pair icons with text labels
- ‚úÖ Provide text alternatives for visual elements
- ‚úÖ Ensure keyboard navigation works
- ‚úÖ Use semantic HTML
- ‚úÖ Test with screen readers

## üì± Responsive Design

### Breakpoints
- **Mobile:** < 768px
- **Tablet:** 768px - 1024px
- **Desktop:** > 1024px

### Mobile Adjustments
- Navigation collapses to hamburger menu
- Cards stack vertically
- Font sizes reduce 10-15%
- Touch targets minimum 44x44px
- Reduced spacing
